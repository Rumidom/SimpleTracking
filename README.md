# SimpleTracking
## Opensource tracking and image stabilization tool

### Depencies:

- Opencv - ( pip install opencv-python )
- Pysimplegui - ( pip install PySimpleGUI )

### How to use:

Install dependencies and then run SimpleTrackingTool.py.
